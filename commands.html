List of all the commands for untitled-bot, separated by category.

Click a category to see the commands!

Rank commands

  • rank - get the rank of yourself or another user.
  • top - display a leaderboard of the highest leveled users in the server.
  • rank-settings - settings related for ranks (such as level up messages) for server owners.
  • rank-role - give users roles for when they level up!
  • rank-roles - list current roles to be assigned on level up.

Utilities and other such commands

  • help - get help with either a specific command or list all commands.
  • prefix - set the bot's prefix for this server.
  • timestamp - get the time a user made their account, a channel was made, etc.
  • info - get information about the server or a user.
  • avatar - get yours or another user's avatar.
  • reverse - reverse text.
  • config - easy config for the bot.
  • profile - get the profile of another user.
  • caps - MAKE SOME TEXT ALL CAPITAL LETTERS
  • lowercase - make some text all lowercase letters.
  • mock - MaKe A mEsSaGe ChAnGe EaCh LeTtEr CaPiTaLiZaTiOn.
  • source - get the source code for the bot.
  • which - get the source code for a particular command.
  • perms - get the server permissions of yourself or another user.
  • rot13 - run some text through an ROT13 converter.

Meta commands for the bot itself.

  • bug-report - report a bug directly to me.
  • discord - get an invite link for the support server.
  • vote - get a link to vote for the bot on Top.GG
  • ping - get the bot's ping
  • uptime - get the amount of time the bot has been online.
  • stats - display some statistics for the bot.

Moderation and logging commands

  • log-channel - set the channel to log events to.
  • add-log - add a log type to the channel.
  • remove-log - remove a log type from a channel.
  • get-log - list all logs currently in use.

Fun commands

  • someone - mention a random user without pinging them.
  • 8ball - simulate an 8ball.
  • ship - ship two users.
  • 20 - roll a twenty-sided die.
