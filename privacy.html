untitled-bot privacy stuff

I take privacy as seriously as the next person, assuming the next person takes privacy as seriously as I do. Which is very.

Above that, I believe that services should be 100% open about what they collect, which is exactly what I'm going to do.

What I store on disk

I'm very open about what I store on users. So open, in fact, that you can actually request your data to be sent to you in a zip file through the bot itself!

Just join the support server (listed above) or any other server with the bot, then execute the <data command.

Back on-topic though, this is what I store on users:

  • Level, XP, balance, and other data sent through commands.
  • Cooldowns for specific commands that use them.
  • Your user ID (it uniquely identifies you to store your data, but the ID is public regardless).
  • The ID of the server you are in.
  • The names of commands users execute, but not the arguments (so "<rank @user" will only log "<rank").
  • Any data stored with commands. The bot makes it somewhat obvious when something is stored on disk.





What I cache

For those who don't know, caching allows bots to store data inside memory which can be used later without sending additional requests to Discord.

Cached entities include:

  • Users (to speed up the leaderboard command as well as rank commands).
  • Text and voice channels (for the music commands and others).
  • Messages (if, and ONLY IF the server administrators setup deleted/edited message logging).
  • Servers (to display the server count and make sure the bot doesn't freeze for some servers).
  • Roles (for giving users a role upon level up).





Am I safe using this bot?

Yes. I don't sell your data, and never will, nor will I share it with anyone.

How do I delete my data?

Contact me on the official support server (listed on the nav bar above) or, if you are the server owner, you can use the "purge" command to remove the bot from the server and delete any data associated with that server.

Your data may also be deleted after about three weeks of not being updated, mostly so it doesn't fill up my disk space :P
