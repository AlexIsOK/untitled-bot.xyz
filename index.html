About untitled-bot

untitled-bot, in all lowercase letters, is a bot I made for fun. It started off as a bot for a server owned by my friend, but I figured I might as well share it. After a while, I submitted it to top.gg, and, after a few weeks, was approved.
Enough about history, let's talk about the bot itself. The bot is structured in such a way where it can be expanded upon very easily, with basic knowledge of Java. It does many things, most of which are utilities or fun commands. The RPG command, version 1.4, is still being made. The commands section of this page lists much more information.








Commands

untitled-bot comes with many commands, all of which come standard.

Ranking
> rank [user @] - get the rank of yourself or another user.
> leaderboard - show a list of the highest ranking users in the server.
> rank-total - show the total amount of xp someone has.
> rank-settings <setting> - set the rank settings (use help rank-settings for a list of rank settings).
> rank-role - give users roles when they level up.
> rank-roles - get a list of roles that are assigned on rankup.
Economy
> daily - get UB$ every day.
> work - work to get UB$.
> steal - steal UB$ from another user.
> bal - list the balance of yourself or another user.
Utilities
> help [command] - get help with a specific command, or display all the commands.
> prefix - set the prefix for the guild.
> status - get the status and some statistics of the bot.
> timestamp <snowflake | user @ | channel #> - get the time a user made their account, a channel was created, or the timestamp of a snowflake.
> uptime - get the time since the bot started
> info - get info about the server or a specific user.
> avatar - get the avatar of yourself or another user.
> bug-report - report a bug directly to me.
> discord - get an invite link for the Discord support server.
> vote - vote for the bot on Top.GG.
> ping - get the ping of the bot.
> inv - list your inventory.
> reverse - reverse a string.
> config - easy configuration for the bot.
> profile - get the profile of another user.
> caps - change a string to all capital letters.
> lowercase - change a string to all lowercase letters.
> alternate-case - MaKe A mEsSaGe ChAnGe EaCh LeTtEr CaPiTaLiZaTiOn.
Moderation
> log-channel - set the channel where things will be logged.
> add-log - add a logging type to the channel.
> remove-log - remove a logging type from the channel.
> get-log - list all logs currently in use.
Fun
> someone - mention a random user without pinging them.
> brainfuck - run brainf*** code.
> 8ball - simulate an 8 ball.
> ship - ship two different server members (user requested).
> 20 - roll a twenty-sided die.
> owo <text> - convewt da text u input to make it vewy kawaii <3
Reactions
> hug - send a hug gif.
> ayaya - I encourage you to not ask questions about this command.
> hide - hide behind a wall.
> dis - short for disappointed, display ಠ_ಠ

Contributions are more than welcome, they're encouraged! Even if you don't know how to program, if you want to suggest things to add to the bot, please join the Discord server :)


As of the newest update, contributors get their own custom background for the `rank` command!








Privacy Pledge

Privacy, especially in today's world, is hard to come by. With some other Discord bots, you might not know what they're storing, much less who can access that data. With untitled-bot, you can see exactly what the source code does, what the moderators log through this bot, and request your data at any time*. The minimal data stored by the bot uses your ID, and while it is identifiable to your Discord account, it is of no practical use to most.

Popular Discord bots, especially those that have been approved by Discord, are almost certainly safe.

Creating a better and safer internet starts by asking the question, "do I really need to store this?"

Contributing

Even if you don't know how to program, contributing ideas or reporting bugs still helps greatly! If you want to contribute ideas, there is a channel specifically for that in the Discord server. You can also create a GitHub account and create an issue for bugs/feature suggestions.

For programmers, specifically Java programmers, you can fork the repository and contribute changes. If you have any questions about contributing, feel free to DM me on Discord.













* the "data" command is only usable once every 24 hours, or, to be more precise, you must wait more than 86400 seconds between data requests. You can only use the command if you are in a server where untitled-bot can read your message. Your data is sent to you in a direct message, it is not shared with anyone else.
