What's your stance on privacy?

Privacy should be a right, something that every service should have. That's why untitled-bot is completely open-sourced and allows you to obtain your data through the "data" command.



Why is this free? What's the catch?

The catch is, once I reach 1,000,000 servers, I will have conquered the planet itself!!!111!
Seriously though: I started this bot for a private server with my friends, then kept adding changes to it. Eventually, I thought "what if other people want to use this bot as well?", so I put the bot on Top.GG (Discord Bot List) and from there it gained around 30 servers in the first few days of it being added!
I'm also doing this to combat and provide an alternative for some other paid-for bots that lock away features that are very easy to make, such as rank roles, starboard, and music players.
Other than that, I don't have any real "reason" as to why I'm doing this. I just do it because I like programming and because making this bot has been a great learning experience for me.



What kind of data do you store on me? How do I delete it?

I store minimal data on users. Members, as described in JDA, are cached in memory, along with guilds (servers), roles, messages, and text channels are cached in RAM for fewer requests to Discord and much faster response time. Those entities are never stored on disk, and are deleted when the bot restarts. For data that is stored on disk and kept after the bot restarts: the bot will keep your user ID (public), servers that you share with the bot, and any user data associated with the bot (such as xp, level, balance, etc.). You can request a copy of your data by using the "data" command.

That's cool and all, but how do I delete my data?

Currently, no command exists to delete your data; however, I will be accepting requests to delete data if you DM me on Discord.



I have an idea for a command, can I add it to the bot?

Yes, absolutely! If you want to program the command yourself, you can do so and create a pull request on the GitHub page.

If you don't know how to program or don't want to, you can join the Discord server and add it to the #suggestions channel.
